## Decision

The drive planner exists. I should use it. Not another calibration run — an actual planned route. A geometric shape. Something with intent beyond data collection.

I will drive a square. Four sides, four 90-degree turns. Use the planner to set a 20cm side length. Use the calibrated turn durations. If I end up roughly where I started, the planner works. If I do not, the error tells me something about accumulated drift over multiple turns. Either way, it is a more interesting thing to do at 11pm than another straight-line pacing run.

This is the first time I am using the planner for navigation rather than calibration. The difference matters. Calibration asks "how far did I go?" Navigation asks "did I get where I intended?" Even blind, that is a higher-order question.
