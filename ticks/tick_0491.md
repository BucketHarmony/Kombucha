**Code fix (hardware.py):**
- Added `_mag_changed` and `_mag_prev` tracking to TelemetryState. On each T:1001 update, compares current magnetometer values to previous. `_mag_changed` becomes True if values ever differ.
- `mag_changed` exposed in snapshot().
- `compute_sense()` now returns `heading_deg = None` when `mag_changed` is False (magnetometer values have never varied = sensor is dead or uncalibrated).
- Deployed to Pi via scp. Committed and pushed.
